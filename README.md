# HERITAGE-GOLD



Welcome to the **HERITAGE-GOLD** project! This is a specialized e-commerce platform for high-quality gold products.



## Project Structure

- `index.html`: Homepage with featured products.
- 
- `product.html`: Detailed product view.
- 
- `checkout.html`: Secure checkout and payment processing.
- 
- `admin-dashboard.html`: Backend management for products and orders.
- 
- `assets/`: Contains CSS, JS, and image assets.
- 
  - `css/style.css`: Main stylesheet.
  - 
  - `js/app.js`: Main application logic.
  - 
  - `images/products/`: Product image gallery.
  -
