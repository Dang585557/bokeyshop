# Heritage Gold - Luxury E-commerce Project

A premium luxury e-commerce platform featuring high-end brands like Gucci, Rolex, and Cartier.

## Project Structure

- `index.html`: Home page
- `shop.html`: Product listing page
- `product.html`: Product details page
- `checkout.html`: Checkout and payment page
- `admin-dashboard.html`: Administrative management panel
- `assets/`: Static assets (CSS, JS, Images, Icons)
- `components/`: Reusable HTML components
- `pages/`: Specific pages for brands, collections, and authentication
- `database/`: JSON data files for products, orders, and users
- `tailwind.config.js`: Tailwind CSS configuration

## Technologies

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (Vanilla JS)
- JSON (Mock Database)
